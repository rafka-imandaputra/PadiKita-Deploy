Padi Kita

Padi Kita adalah output ketiga tim Orbit di ajang TSDN 2022. Temuan Dashboard Analytics terkait produksi padi ditindaklanjuti dengan pembuatan machine learning untuk mendeteksi penyakit padi, dan model ini diimplementasikan menjadi website Padi Kita.

Kedepannya website ini dapat dikembangkan dengan membuat fitur realtime klasifikasi agar user hanya perlu mengarahkan kamera langsung ke padi.

Tim Orbit SMKN 4 Bandung
